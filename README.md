# Data Science Internship Project

This repository contains my complete work for a Data Science internship project.

## Project Overview
The project involves end-to-end data analysis on a restaurant dataset, including data exploration, descriptive analysis, feature engineering, predictive modeling, and data visualization.

## Tasks Covered
- Level 1: Data Exploration, Descriptive Analysis, Geospatial Analysis
- Level 2: Service Analysis, Price Range Analysis, Feature Engineering
- Level 3: Predictive Modeling, Customer Preference Analysis, Data Visualization

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

Each notebook includes code, visualizations, and clear explanations for insights derived.
